Frequently Asked Questions



What is GitStarter?



GitStarter is an open source investment app made by and for developers. Anyone can check out Github projects and buy or sell them.



What is the purpose of GitStarter?



GitStarter allows individuals to invest in projects developed by programmers who have a passion for computer science and have a knack for app development. This is a platform that invests in these open-source projects so we can continue to move the world forward.



What are the different features on GitStarter?



Any user on GitStarter can buy and sell Github repositories and track the progress of their investments in terms of the number of commits and activity that the contributors that own the project has put in.



How can I invest in a project?



A user can start by searching for the Github project that he or she would be interested in investing in and click buy to own a share of the project and track their investments. A user can also sell a project and receive any dividends returned.
